== Description ==

The two together is an awsome combination for any merchant looking for an eCommerece pressence, without any finiacial load.
*   "WooCommerce" is an open source application
*   "Velocity" is offering payment.



== Installation ==

1. Ensure you have latest version of WooCommerce plugin installed (WooCommerce 2.0+)
2. Unzip and upload contents of the plugin to your `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress

== Configuration ==

1. Visit the `WooCommerce > Settings > Checkout` tab.
2. Click on *Velocity* to edit the settings. If you do not see *Velocity* in the list at the top of the screen make sure you have activated the plugin in the WordPress Plugin Manager.
3. Enable the Payment Method, name it `Credit Card 
4. Add in your `MerchantProfileid` and `Workflowid` as provided by the Velocity Team.
5. Choose if you want to show the `Velocity` Logo to the customer
7. Click Save.
